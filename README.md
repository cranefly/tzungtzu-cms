# 安装tzungtzu cms
由于系统没有一键安装，所以需要手动安装

1）链接数据库

修改tzungtzu/controllers/config/database.php的对应参数，使其链接数据库

2）在数据中执行doc/tzungtzu.sql,默认的管理员密码与数据库中创建的用户名相同
进入后台可修改管理员密码

3）后台访问地址 域名/webmaster
